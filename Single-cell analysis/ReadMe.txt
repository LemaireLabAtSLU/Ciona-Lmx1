This repository contains the R scripts to perform the single-cell analysis of datasets published by Cao and colleagues in 2019 from the initial gastrula to mid-tailbud stage (GEO accession number GSE131155). The reads were remapped on the KY21 gene model using CellRanger 7.0.1. The analysis was performed in R version 4.5.1 with Seurat version 5.4.0. The version of the other packages used in the analysis is indicated in sessionInfo().txt of the different folders. No non-standard hardware is required for the analysis. R and Seurat pacakge installation can be found on the following website https://www.r-project.org/ and https://satijalab.org/seurat/articles/install respectively. The installation should take typically a few minutes.

Intructions: 
The filtered gene expression matrices were first analyzed using the r script of the iGmT directory to identify and isolate the neural cells. The file iGmT.tissue.types.csv contains the correspondence between the clusters and the identified cell types. iGmT_DEG_combined_res1.txt contains the differentially expressed genes for the different clusters. ciona10stage.cluster.upload.new.txt provides the tissue identity of cells mapped with the KH2012 gene model and published by Cao et al., 2019.
The neural cells are then subsetted as a new Seurat object and saved as an RDS file. This file is then used to run the script NS_analysis.r (NS directory) to identify the b-lineage derived cells, which are subsetted and saved as an rds file. The files NS.DEG_combined_res2.4.txt and NS.cell.types.csv contain the differentially expressed genes for the neural clusters and the correspondence between the clusters and the identified neural cell types, respectively. 
The b-lineage neural cells were then analyzed, and their pseudotime trajectory was computed with the r scripts of the blineage directory. blineage_Association.test_metrics_pt_npbb_consecutive.txt and blineage_clusterLabels_lineage3.txt contain the differentially expressed genes along the trajectories and their cluster numbers for the third lineage, corresponding to posterior dorsal neural tube cells.
The scripts should run in a few hours.
